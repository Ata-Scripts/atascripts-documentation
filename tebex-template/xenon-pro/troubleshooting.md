# Troubleshooting

#### Theme looks broken or unstyled

CSS files were likely not uploaded, or were uploaded under the wrong asset type.

**Fix:** Go to Assets, verify all CSS files from `assets/styles` are present and uploaded as `CSS` type.

***

#### Buttons, animations, or interactions not working

JavaScript files are missing or uploaded incorrectly.

**Fix:** Verify all files from `assets/scripts` are present and uploaded as `JavaScript` type.

***

#### Images are missing

**Fix:** Verify all files from `assets/imgs` are uploaded as `Image` type.

***

#### Layout still looks like the old template

You may have missed replacing one or more HTML files, or forgot to publish.

**Fix:** Double-check all 10 HTML files were replaced, then click **Publish** again.

***

#### Schema options not showing in Preview

This is expected behavior - Schema changes don't always render in Preview.

**Fix:** Click **Publish**. The schema will apply correctly on the live store.

***

#### Accidentally deleted `quote.html` or `category/tiered.html`

These files cannot be restored manually inside the Tebex editor.

**Fix:** Create a brand new custom template using the `Exo` base
