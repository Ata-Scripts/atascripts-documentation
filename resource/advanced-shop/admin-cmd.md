# Admin CMD

Ata Fishing includes several admin commands to manage player fishing levels and XP:

| Command                                 | Description                                               |
| --------------------------------------- | --------------------------------------------------------- |
| /fishing\_add\_level \[ID] \[AMOUNT]    | Add levels to a player                                    |
| /fishing\_add\_xp \[ID] \[AMOUNT]       | Add XP to a player                                        |
| /fishing\_remove\_level \[ID] \[AMOUNT] | Remove levels from a player                               |
| /fishing\_remove\_xp \[ID] \[AMOUNT]    | Remove XP from a player                                   |
| /fishing\_set\_level \[ID] \[AMOUNT]    | Set a player's level                                      |
| /fishing\_information \[ID]             | View a player's fishing stats (works without ID for self) |
